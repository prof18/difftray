# Naming Notes

## Working Name

Difftray

## Rationale

Difftray suggests a desktop tray of diffs waiting for review. It fits the product shape:

- local desktop app
- multiple projects
- review queue
- file-level completion

It avoids the crowded "patch management" space around names beginning with `Patch`.

## Names Rejected

### Patchmark

Rejected because it is already used in an adjacent code diff/review context.

### Patchwise

Rejected because it is already used for kernel patch review tooling and appears in active package/project listings.

### Differ

Rejected because it is already used by active software products and has poor searchability.

### DiffPilot

Rejected because it is already used by a VS Code code review tool.

### DiffScope

Rejected because it is already used by Git analysis tooling.

## Availability Notes

Initial search found no obvious active product, npm package, PyPI package, crates package, or trademark result for `Difftray`.

One old archived Bitbucket repository named `difftray` appears in Software Heritage's Bitbucket archive. This is noted but not currently considered a blocker.

Before public release, re-check:

- GitHub repository availability
- npm package availability
- app/domain availability
- Apple app name availability
- trademark search
