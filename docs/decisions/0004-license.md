# Decision 0004: License

## Status

Accepted

## Decision

Use the Apache License 2.0 for the eventual open-source release.

## Context

Difftray is intended to become an open-source desktop developer tool. The project should be easy for individual developers and companies to adopt, fork, and contribute to.

Apache-2.0 matches the maintainer's usual open-source preference and gives clearer patent and contribution terms than MIT.

## Consequences

Positive:

- Widely understood.
- Low adoption friction.
- Explicit patent grant.
- Patent retaliation clause.
- Clearer contribution and redistribution terms than MIT.

Negative:

- Longer and more formal than MIT.
- Does not require downstream modifications to be open-sourced.

## Notes

Reconfirm before public launch, but Apache-2.0 is the working license and is already included in the repository.
